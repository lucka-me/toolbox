# 简繁转换代码生成器
从简繁体词汇列表生成对应的MediaWiki简繁转换代码列表。

## Usage

1. 准备简体词汇列表`zhs.txt`及繁体词汇列表`zht.txt`
2. 将简体词汇列表`zhs.txt`放入百科页面，利用百科的简繁转换生成简转繁列表`zhs_zht.txt`
3. 同样地，获得繁转简列表`zht_zhs.txt`
4. 运行生成器，获得基于简体的转换代码列表`output.txt`

## Requirements
### Environment and Packages
  * Python 3
