# CSS
个人用 CSS，为 Lucka 的网页项目提供统一的**基础**样式，减少各仓库中的重复代码并集中管理基础样式。

可由[此处](https://lucka.moe/toolkit/css/common.css)引用。

## Functions
提供一系列颜色、文字样式、边框、排版（Flexbox）、对齐、边距、以及统一的物件样式（如按钮、输入框）等。

将随需求而不断扩充内容。
